# EchoME

EchoME is an AI-first portfolio experience built for the AI WebForge hackathon.

It behaves like a digital personality instead of a static resume:

- Visitors choose a mode (Recruiter, Builder, Judge, Curious Human)
- Command input reshapes project emphasis in real time
- Signature feature "Instant Portfolio Recomposer" rewrites your strongest proof for any prompt
- Live Lab demonstrates your thought process on demand

## Stack

- React 19
- TypeScript
- Vite

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Core Experience Map

- Hero: adaptive mode + command bar
- About: interactive narrative framing
- Projects: dynamic ranking based on prompt intent
- Signature Recomposer: paste a brief, get tailored fit summary
- Skills: capability graph grouped by outcomes
- Live Lab: real-time strategy output from user prompt
- Contact: collaboration-first call to action

## Hackathon Pitch

EchoME reframes portfolios from static documents into responsive product experiences.
The demo highlights one key value proposition: relevance on demand.
